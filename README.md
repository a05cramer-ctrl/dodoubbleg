# Dogg Coin

Ett React-projekt byggt med Vite och TypeScript.

## Kom igång

1. **Installera Node.js** (om du inte redan har det):
   - Ladda ner från [nodejs.org](https://nodejs.org/) eller använd `nvm`

2. **Installera beroenden:**
   ```bash
   npm install
   ```

3. **Starta utvecklingsservern:**
   ```bash
   npm run dev
   ```

4. Öppna [http://localhost:5173](http://localhost:5173) i webbläsaren.

## Tillgängliga kommandon

- `npm run dev` – Startar utvecklingsservern
- `npm run build` – Bygger för produktion
- `npm run preview` – Förhandsgranskar produktionsbyggen
- `npm run lint` – Kör ESLint
